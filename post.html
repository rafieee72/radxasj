Belajar HTML & CSS

Artikel ini akan membahas cara membuat halaman web pertama Anda menggunakan HTML dan CSS. HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web, sementara CSS digunakan untuk mengatur gaya tampilan halaman tersebut.

Di artikel ini, Anda akan belajar tentang tag HTML dasar, cara membuat halaman web, dan cara memberi gaya menggunakan CSS. Mari mulai dengan membuat sebuah file HTML pertama Anda.

Langkah 1: Membuat Struktur Dasar HTML

Langkah pertama adalah membuat file HTML. Berikut adalah contoh kode HTML dasar:

<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama Saya</title>
</head>
<body>
    <h1>Selamat datang di Website Saya</h1>
    <p>Ini adalah website pertama saya!</p>
</body>
</html>
